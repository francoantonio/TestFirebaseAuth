## Proyecto de prueba con Firebase Auth

Para utilizar el proyecto se tiene que crear el archivo /config/config.js
```js
export const firebaseConfig = {
	apiKey: '-----',
	authDomain: '--------',
	projectId: '---------',
	storageBucket: '-----',
	messagingSenderId: '-------',
	appId: '---',
	measurementId: '---',
};
```
Remplazando los valores por los suyo en sdk de firebase
